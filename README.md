# prototype-LLM--Gambit
Gambit-LLM: Natural Language Game Theory Engine
This project is a prototype designed to integrate Large Language Models (LLMs) with the Gambit software suite. It bridges the gap between descriptive game scenarios and formal mathematical representations (Game Trees and Strategic Forms).

Key Features
Prompt Correction: Automatically refines vague or inconsistent game descriptions into logically sound prompts.

Game Tree Generation: Visualizes extensive-form games directly from text.

Payoff Matrix Synthesis: Generates strategic-form tables for simultaneous-move games.

Web-Based Interface: A lightweight, accessible UI for rapid prototyping.
